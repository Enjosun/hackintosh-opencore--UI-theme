# hackintosh-opencore--UI-theme

下载此附件中Resources文件夹替换EFI/OC/同名文件夹

下载此附件OpenCanopy.efi 替换EFI/OC/Drivers/下同名文件，并确认并在config.plist加载该驱动

EFI/OC/Drivers/中必须存在CrScreenshotDxe.efi 并在config.plist加载该驱动
